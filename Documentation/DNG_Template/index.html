DNG Template documentation

Version: 1.1 | Last update: 01/15/2018

Thank you for purchasing DNG template on ThemeForest!

The documentation will help you know the structure of template and guide you about the common functions. If what you need can’t be found here, please contact us for further help.

Our email address is support@dnngo.net

Files Structure

Once you download the package, you will find the following folders:

  • HTMLTemplate Files
    • CSSTemplate CSS
      • componentCSS of components
      • demosCSS of demo pages
      • elementsCSS of elements
      • theme-colorCSS of accent colors
      • theme.cssMain CSS of template
      • blog.cssCSS of blog page
      • dark.cssCSS of dark mode
      • dnngo-font-icon.cssCSS of font icons
      • footer.cssCSS of footer
      • header.cssCSS of header
      • page-title.cssCSS of page title
      • portfolios.cssCSS of portfolio page
      • sidebar.cssCSS of sidebar
      • widget.cssCSS of widget
    • fontsFont folder
    • icon-svgIcons in Svg format
    • insetImages folder
    • jsJavacripts folder
    • vendorJavacripts plugins folder
  • DocumentationDocumentation
  • PSDDownload URLs for PSD files

HTML Structure

The template is based on Bootstrap 3 Framework - http://getbootstrap.com

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases.

Below is a sample coding structure:


<!DOCTYPE html>
<html lang="en">
  <head>
    <!-- Page meta, page css, page js files, page title etc... -->
  </head>
<body>   

  <!-- Theme Main -->
  <div class="theme-main">
    <div id="wrapper">

      <!-- Header -->
      <header>
      ...
      </header>   

      <!-- Mobile Header -->
      <div class="visible-xs visible-sm mobile-header">
      ...
      </div>   

      <!-- Content -->
      <section id="page-content">
      ...
      </section>  

      <!-- Footer -->
      <footer class="footer-box footer-01">
      ...
      </footer>  
      
      <!-- Back Top -->
      <div id="to-top" class="backtop01"><span></span></div>

    </div>        
  </div>

  <!-- Scripts -->
          
  </body>
</html>

Layout Settings

The template supports 2 different kinds of layout modes: Boxed & Wide Layout. And Wide Layout is the default option, if you want to use the Boxed version, please add the following class in<html>, then the layout style will be changed to boxed version.

<html class="boxed">

Logo Settings

The default website Logo is in the #header container.


<div class="Logobox">
  <a title="Logo" href="#"><img src="inset/logo.png" srcset="inset/logo-2x.png 2x" alt="Logo"></a>
</div>

Retina Logo If you want to display high-resolution retinal with retina logo on your website, then you just need to put the same directory to the high-resolution file whose name is logo-2x.png.

Roll Menu Logo Settings

You can use the following code to set the logo of roll menu


<div class="FixedLogoPane">
  <a title="Logo" href="#"><img src="inset/logo.png" srcset="inset/logo-2x.png 2x" alt="Logo"></a>
</div>

Favicons Icons

You can add a Favicon to your Website by using the following code:


  <link href="inset/favicon.ico" rel="shortcut icon">

Page Loading Transitions

You can show interactive loaders to your visitors while the site pages load

First, you need to put the following code to <head></head>


  <link href="css/component/loaders.css" rel="stylesheet" type="text/css">

Then, put the following code to the top of <body></body>


  <!--Page loaders-->
  <div class="page-loaders">
    <div class="loaders">
      <div class="loader-inner ball-clip-rotate-multiple loader-accent">
        <div></div>
        <div></div>
      </div>
    </div>
  </div>

More loading animations, please refer Page-loaders.html file

Dark Scheme

You can completely change the Contrast of the Website to Dark

First, you need to put the following code to<head></head>


  <link href="css/dark.css" rel="stylesheet" type="text/css">

The add the following classes to <html>

<html class="dark">

Fonts

This theme is using Google Fonts Library, check the head of website, you will find the tag here

<!--Google Fonts-->
  <link href="https://fonts.googleapis.com/css?family=Raleway:100,400,600" rel="stylesheet" type="text/css">

Want to change the font? Please go to http://www.google.com/webfonts to choose one font and use the generated code.

Google Map

You can create unlimited contact maps on your site, and here are the steps:

First, copy the following code to the end of <body></body>


<script src=" vendor/map/google.map.js" ></script>

Then copy the following code to content area


<div class="google-map contact-map" 
  data-Scrollwheel="false" 
  data-icon='inset/homepage03/home03_map.png' 
  data-address="Bear city, ny" 
  data-zoom='14' 
  data-content='<h6>COMPANY NAME</h6><p>16th floor, Downing Building, Jianshe Road,</p><p>Wuhan, China.</p><p>123-456-789</p>'
  data-styles='[ { "featureType": "water", "elementType": "geometry", "stylers": [ { "color": "#e9e9e9" }, { "lightness": 17 } ] }, { "featureType": "landscape", "elementType": "geometry", "stylers": [ { "color": "#f5f5f5" }, { "lightness": 20 } ] }, { "featureType": "road.highway", "elementType": "geometry.fill", "stylers": [ { "color": "#ffffff" }, { "lightness": 17 } ] }, { "featureType": "road.highway", "elementType": "geometry.stroke", "stylers": [ { "color": "#ffffff" }, { "lightness": 29 }, { "weight": 0.2 } ] }, { "featureType": "road.arterial", "elementType": "geometry", "stylers": [ { "color": "#ffffff" }, { "lightness": 18 } ] }, { "featureType": "road.local", "elementType": "geometry", "stylers": [ { "color": "#ffffff" }, { "lightness": 16 } ] }, { "featureType": "poi", "elementType": "geometry", "stylers": [ { "color": "#f5f5f5" }, { "lightness": 21 } ] }, { "featureType": "poi.park", "elementType": "geometry", "stylers": [ { "color": "#dedede" }, { "lightness": 21 } ] }, { "elementType": "labels.text.stroke", "stylers": [ { "visibility": "on" }, { "color": "#ffffff" }, { "lightness": 16 } ] }, { "elementType": "labels.text.fill", "stylers": [ { "saturation": 36 }, { "color": "#333333" }, { "lightness": 40 } ] }, { "elementType": "labels.icon", "stylers": [ { "visibility": "off" } ] }, { "featureType": "transit", "elementType": "geometry", "stylers": [ { "color": "#f2f2f2" }, { "lightness": 19 } ] }, { "featureType": "administrative", "elementType": "geometry.fill", "stylers": [ { "color": "#fefefe" }, { "lightness": 20 } ] }, { "featureType": "administrative", "elementType": "geometry.stroke", "stylers": [ { "color": "#fefefe" }, { "lightness": 17 }, { "weight": 1.2 } ] }]'>

Options:

  • data-Scrollwheel - set scroll wheel to zoom map
  • data-icon - set icon
  • data-address - set location address
  • data-zoom - set zoom size
  • data-content - set popup content of icon
  • data-styles - set map style. You can gain more styles from site https://snazzymaps.com/explore.
