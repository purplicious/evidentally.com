Introduction

Create stunningly beautiful countdowns in a few simple clicks with Soon. Soon contains a diverse set of counter types and comes with a visual builder so you can easily configure your own unique counters.

Package Contents

  • A 'bin' folder containing the library itself.
  • A 'docs' folder containing detailed documentation and examples.
  • A 'src' folder containing the source files.
  • An 'extras' folder containing the presets and the visual builder.

You'll probably only need the source files if you are an experienced webdeveloper and want to modify the package.

Setup

Soon is easy to setup by adding both soon.min.css and soon.min.js (found in the 'bin' folder) to your website.

  1. First copy the contents of the bin folder to your site folder.

  2. Then copy the following HTML snippet in the <head> of your webpage.

    <link href="css/soon.min.css" rel="stylesheet">

  3. Then copy the following snippet and place it just before the closing </body> tag.

    <script src="js/soon.min.js"></script>

Make sure the paths in the above code snippets match the location of the CSS and JS files.

The Soon JavaScript files run automatically, so you don't have to worry about it.

Using Soon with a module loader

If you are using a module loader it's adviced to use the soon.module.js file. This version of Soon does not contain the kickstart feature, does not expose the jQuery API and has no embedded polyfills (requestAnimationFrame, bind, classList, indexOf) also, it has not been minified.

Browser Support

Soon works on all modern mobile and desktop browsers.

The following older browsers are supported:

  • Safari (Windows): Safari on Windows is no longer being maintained by Apple and therefor is currently not supported. Soon works great on the OSX version of Safari.
  • IE9: lacks support of animations and circular counter
  • IE8: will fall back to default text counter for performance and compatibility reasons
